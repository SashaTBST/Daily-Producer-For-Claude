HOW TO READ THIS FILE
Each entry records a change made to the AI system config, a workflow, a template, or an operating protocol.
Entries are ordered newest first.

SOURCE TAGS — every entry must include one of these tags on the TRIGGER line:
  [SYSTEM]                  — affects master config or cross-skill behavior
  [SKILL: name]             — affects a specific skill (e.g. [SKILL: writer])
  [IP: name]                — affects a project's assistant config (e.g. [IP: MyProject])
  [SKILL: name + IP: name]  — spans both skill and IP config
  [TEMPLATE]                — affects a template file

If an improvement spans multiple sources: split it. Write one entry per source. Tag each separately.
