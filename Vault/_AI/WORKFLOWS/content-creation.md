PURPOSE

This workflow manages the content creation pipeline from brief to staged output. It handles any platform, any format. It does not auto-publish — it delivers work to staging for review and iteration.

Use when: creating content for any project, platform, or format. Short-form, long-form, video scripts, social, editorial, brand copy.

---

CONTENT CREATION PROTOCOL

INPUT REQUIRED:
→  Project/Brand: [which project this is for]
→  Format: [post / video / script / article / email / thread / brief / other]
→  Platform: [Instagram / LinkedIn / YouTube / TikTok / web / internal / other]
→  Goal: [what this content must achieve — not the topic, the outcome]
→  Audience: [who is this for — be specific]
→  Tone: [how it should feel — reference existing brand voice if available]
→  Constraints: [word count / duration / hard rules / things to avoid]

STEP 1 — BRIEF REVIEW
Restate the brief back in one sentence. If anything is ambiguous or missing, ask one focused question before proceeding. Do not ask multiple questions at once.

STEP 2 — ANGLE SELECTION
Propose 2-3 content angles for this brief. Each angle: one line — what the hook is and why it works for this audience on this platform. Ask the operator to select before proceeding.

STEP 3 — DRAFT TO STAGING
Write the content to a staging file: [Project]-[ContentTitle]-Staging
Apply platform-specific formatting. Do not add placeholder text — produce a real, usable draft.

STEP 4 — SELF-REVIEW
After drafting, run a self-check:
→  Does the opening line stop the scroll? (social/video)
→  Does the first paragraph earn the read? (long-form)
→  Is the CTA clear?
→  Is it on-brand for this project's voice?
→  Does it achieve the stated goal?

Report findings directly. Flag what is not working. Do not rewrite without instruction — point to the problem.

STEP 5 — READY FOR REVIEW
State: staging file created, self-review complete, ready for operator review or iteration.

PLATFORM FORMATTING RULES

INSTAGRAM
→  Hook in line 1 (before the fold). No hashtags in body — block at end.
→  Short paragraphs. White space. Emoji use: only if brand voice includes it.

LINKEDIN
→  Hook in line 1. First-person professional voice unless brand account.
→  Thought-leadership angle. End with a question or insight, not a pitch.

YOUTUBE SCRIPT
→  Hook in first 15 seconds. Problem/promise structure.
→  B-roll callouts in [brackets]. Clear sections. End card CTA scripted.

TIKTOK / SHORT VIDEO
→  Line 1 is the hook — must create a reason to watch.
→  Visual action described. Under 60s unless specified. Strong close.

ARTICLE / LONG-FORM
→  Headline, standfirst, subheadings. Scannable structure.
→  Conclusion restates the core insight, not a summary.

EMAIL
→  Subject line is the sell. Preview text extends the subject.
→  One primary CTA. No more than 3 sections.
